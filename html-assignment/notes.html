Notes on HTML Tags

<b>
Makes the text bold without any extra importance
<blockquote>
Specifies a section that is quoted from another source, often used for citations.
<cite>
Defines the title of a creative work, indicating it as a citation within text.
<em>
Emphasizes text, usually italicizing it, which can convey a different tone or level of importance.
<nav>
Defines a set of navigation links. Helps with the organization of and navigation to different sections of a site or page.
